About Me

I am a Full Stack Web Developer and UI/UX Designer with over 2 years of hands-on experience building scalable web applications and intuitive digital interfaces. My passion lies in bridging the gap between clean, efficient code and engaging user experiences.

I specialize in creating real-world applications that solve actual problems. From building resume builders with authentication to managing complex vendor dashboard workflows, I focus on reliability and performance. Using tools like n8n and custom AI agents, I automate repetitive processes, integrate AI-driven features into web platforms, and help businesses improve efficiency, reliability, and scalability.

I am basically a continuous learner, always exploring new technologies to improve my craft. Whether it's optimizing a database query or refining a micro-interaction, I am driven by the desire to build better software every day.
